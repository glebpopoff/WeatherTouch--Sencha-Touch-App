WeatherTouch - Hourly Weather Forecast Mobile Web App

Built using Sencha Touch Mobile Web App Framework and Google AppEngine

The app demonstrates capabilities of the Sencha Touch JavaScript framework for building mobile Web Apps.
View AppGet Source
  • HTML5 Geolocation
  • Mobile Forms
  • JSONP and AJAX Proxy
  • Data Stores and Multiple Model Associations (BelongsTo)
  • Pull to Refresh Data Reloading
  • CSS3 Animation
  • Multi Card Layout Screens
  • Scroll Touch Events and List Views
  • Pop-up Overlays and Pop-up Messages
  • Google Android and Apple iOS
The app was built using Google App Engine in Python and Sencha Touch framework. If you're using Chrome or on a touch-enabled device, you can view the app here.

The app uses the WeatherBug JSON API for weather feed and Google Maps for reverse geo lookups. It has screen screens (shown below) and a form that allows you to enter location information (state, city, zip). The location data is stored in App Engine's DataStore. All source (js and python) is available on the GitHub.
